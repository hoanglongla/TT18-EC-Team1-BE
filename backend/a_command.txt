
1. copy .env.example => .env
Cấu hình db nếu mysql có thay đổi
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=econail
DB_PASSWORD=

2. Thêm cấu hình máy chủ ảo vào vào file C:\xampp\apache\conf\extra\httpd-vhosts.conf
cấu hình localhost

<VirtualHost *:80>
    DocumentRoot "C:\xampp\htdocs\Team6-BE\backend\public"
    ServerAdmin admin@localhost
    ServerName econail.localhost

    <Directory "C:\xampp\htdocs\Team6-BE\backend">
       Options Indexes FollowSymLinks
       AllowOverride all
    </Directory>
</VirtualHost>
Sau đó nhớ reset lại xammp php và db

2. Cài laravel
composer install (chưa có thì cài)
php artisan key:generate
